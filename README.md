# REAL8 Governance

Project-level governance, strategy, and proposal documents for [REAL8](https://real8.org/en/) — a Stellar-native asset launched in 2018 with a mission of financial inclusion in underserved communities.

This repository is separate from REAL8's code repositories (wallet, bridge, gateway, etc.) and focuses on documents that guide *how* the project is run and *where* it is going, rather than *what* it is built from.

## Contents

- [**`scf-build-award-2026/`**](./scf-build-award-2026/) — Materials for the 2026 Stellar Community Fund Build Award submission, including the Community Agents Network technical architecture.

_Future additions will include multisig policy, agent-network contracts, grant reports, and community proposals._
